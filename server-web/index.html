サーバー紹介

「かまきりの爪サーバー」へようこそ！このサーバーは、RTMのサーバーです 仲間と協力して巨大建築に挑戦するもよし、一人で建築するのもよし、のんびり暮らすもよし。 あなただけのマイクラライフをここで見つけてください！

  • サーバーのジャンル: RTM・クリエイティブ
  • 主な特徴:
    • 24時間365日稼働
    • 独自の経済システムでプレイヤー間取引が活発
    • 初心者サポートが手厚い
    • 定期的なイベント開催 (建築コンテストなど)

サーバーのルール

みんなが気持ちよく遊ぶために、以下のルールを守ってください。

  1. 荒らし・窃盗の禁止: 他のプレイヤーの建築物やアイテムを許可なく破壊・窃盗する行為は厳禁です。
  2. チート・不正ツールの使用禁止: X-Rayや自動化MODなど、ゲームを有利に進める不正なツールの使用は禁止します。
  3. 暴言・迷惑行為の禁止: チャットやVCでの暴言、他人を不快にさせる言動はやめましょう。
  4. 過度な負荷をかける行為の禁止: サーバーに負荷をかけるような大規模な回路や装置を作る際は、事前に運営に相談してください。
  5. その他: 運営の指示には必ず従ってください。ルールは予告なく変更される場合があります。

参加方法

  1. Minecraftを起動し、「マルチプレイ」を選択します。
  2. 「サーバーを追加」をクリックします。
  3. サーバー名に分かりやすい名前（例：かまきりの爪サーバー）、サーバーアドレスに「a-minecraft.ddo.jp」を入力します。
  4. 「完了」を押し、サーバーリストから追加したサーバーを選択して「サーバーに接続」をクリックします。
  5. ようこそ！サーバーロビーに接続されます。冒険を始めましょう！
